### 一、相关技术
- 前端：`Vue`、`HTML` / `CSS` / `Javascript`。
- 后端：`Springboot`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 前端：`node`、`vue-cli` 、`npm`  。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 首页、3. 兼职信息、4. 留言反馈、5. 个人中心、

管理员功能：1. 登录、2. 个人中心、3. 用户管理、4.企业管理、5. 兼职信息管理、6. 职位申请留言板管理、7. 留言板管理、8. 系统管理、
